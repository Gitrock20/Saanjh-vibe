# ABSTRACT

### AIM
The aim of this project is to develop a modern, full-stack web-based e-commerce application named Saanjh for selling luxury hand-poured candles and fine jewelry products through a premium, responsive digital platform.

### SCOPE
The application enables customers to browse products, manage shopping carts, apply promo coupon discounts, place orders, make secure online payments, track order processing statuses, and securely log in using Firebase services. The project is designed for high performance, Search Engine Optimization (SEO), and scalable commercial deployment.

### OBJECTIVES
* To provide a premium and seamless online shopping experience.
* To manage product catalogs and inventories efficiently.
* To implement secure user and administrator authentication.
* To maintain customer orders and transaction histories in real time.
* To provide automated promo code validation and discount calculations.
* To integrate secure online payment processing.
* To develop an administrative dashboard for inventory, order status updates, and coupon management.

### FORMAL DESCRIPTION
Saanjh is a full-stack web e-commerce application developed using React 19, TanStack Start, and Firebase cloud services. The application provides storefront functionalities such as user registration and login, dynamic category filters (focusing 70% on candles and 30% on jewelry), shopping cart drawer operations, coupon discount codes, checkout validation, Razorpay online payments, order success redirects, customer order history logs, and automated transactional emails.

The backend infrastructure utilizes serverless server functions for secure transaction verification, Firebase Authentication for user accounts management, Cloud Firestore for NoSQL real-time databases, and the Resend API for automated email delivery. The platform is designed with a modern, isomorphic architecture to ensure fast load speeds, maintainability, and clean code separation.

### TECHNOLOGIES USED
* React 19 (Frontend Interface)
* TanStack Start & TanStack Router (Full-Stack Framework & File-Based Router)
* TypeScript (Type-Safe Logic)
* Tailwind CSS v4 (Styling & Layout Animations)
* Firebase Authentication (User Session Control)
* Cloud Firestore (NoSQL Real-Time Database)
* Razorpay Payment Gateway (Online Checkout Integration)
* Resend API (Transactional Email Dispatch)
* Vite 7 (Asset Bundling & Development Server)
